# @digta/openprotocol

Atlas Copco **Open Protocol** client for tightening tools and torque
controllers (MID message framing over TCP).

> **Not** Google Protocol Buffers. This is the industrial assembly-tool
> protocol used by Atlas Copco / Bosch Rexroth controllers.

> **Status: placeholder.** Implementation in progress — no runnable API yet.

## Install

```bash
npm i @digta/openprotocol
```
